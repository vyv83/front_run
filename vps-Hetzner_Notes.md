# Заметки по настройке сервера Hetzner Cloud

### 1. Подключение (Важные нюансы)
*   **Пользователь:** По умолчанию Hetzner прописывает SSH-ключ для пользователя **`root`**, а не `ubuntu` (как написано во многих инструкциях).
*   **IP сервера:** `46.62.144.105`
*   **Команда для входа:** `ssh -i ~/.ssh/id_ed25519 root@46.62.144.105`

### 2. SSH Ключи
*   Ключ был сгенерирован на Mac по пути: `~/.ssh/id_ed25519`
*   Тип ключа: `ed25519` (современный, быстрый).

### 3. Быстрый доступ (Alias)
Теперь тебе не нужно помнить IP и пользователя. Я настроил всё на твоем компьютере.
Просто набери в терминале:
```bash
ssh hetzner
```
Это автоматически подключит тебя как `root` с использованием нужного ключа.

### 4. Окружение
*   Папка проекта: `~/collector`
*   Виртуальное окружение: `~/collector/venv`
*   Активация: `source ~/collector/venv/bin/activate`
